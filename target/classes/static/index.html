Bienvenido a nuestra pequeña página web dedicada a la metodología BEM (Block Element Modifier). BEM es una convención de nomenclatura para componentes en el desarrollo de front-end que facilita la escritura de CSS de manera clara y mantenible. Al estructurar el código CSS en bloques, elementos y modificadores, BEM ayuda a crear componentes reutilizables y escalables, mejorando la colaboración entre desarrolladores y diseñadores.

En este caso, el menú fue diseñado implementando la metodología BEM

Primero se establece el contenedor nav que sera la barra de navegación

Luego por debajo iran los componentes principales siguiendo la estructura nav__componente, en este caso el titulo y el menu

Luego tenemos cada enlace de la barra de navegacion, siguiendo la estructura nav__componente-item
